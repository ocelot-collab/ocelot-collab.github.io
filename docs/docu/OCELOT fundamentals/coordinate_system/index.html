Ocelot Coordinate System

The coordinate system in Ocelot follows these conventions:

(x,x′=pxp0),(y,y′=pyp0),(τ=cΔt,p=ΔEp0c)\left (x, \quad x' = \frac{p_x}{p_0} \right), \qquad \left (y, \quad y' = \frac{p_y}{p_0} \right), \qquad \left (\tau = c\Delta t, \quad p = \frac{\Delta E}{p_0 c} \right)(x,x′=p0​px​​),(y,y′=p0​py​​),(τ=cΔt,p=p0​cΔE​)

Definitions​

  • τ=ct−sβ0\tau = c t - \frac{s}{\beta_0}τ=ct−β0​s​: Longitudinal coordinate of the particle.
  • β0=v0c\beta_0=\frac{v_0}{c}β0​=cv0​​: normalized velocity.
  • sss: Independent variable representing the distance along the beamline (i.e., the path length of the reference particle).
  • v0v_0v0​ and p0p_0p0​: Velocity and momentum of the reference particle, respectively.
  • ttt: Time at which a particle reaches position sss along the beamline.

For the reference particle:​

  • τ=0\tau = 0τ=0 for all sss.

For other particles:​

  • τ<0\tau < 0τ<0: The particle arrives earlier than the reference particle.
  • τ>0\tau > 0τ>0: The particle arrives later than the reference particle.

Energy relation:​

  • ΔE=E−E0\Delta E = E - E_0ΔE=E−E0​, where E=γm0c2E = \gamma m_0 c^2E=γm0​c2 is the total energy of the particle.
info

The Ocelot coordinate system is not canonical. For a canonical system, the sign of the τ\tauτ coordinate must be reversed. In that case, the system becomes identical to the MAD coordinate system.
