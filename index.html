OCELOT

A Python-based Framework for Accelerator Design and Simulations

Getting Started

Accelerate Development

Ocelot is designed for studying FEL and storage ring-based light sources, providing everything you need to kickstart your project and save time.

Focus on Physics

Written in Python, Ocelot is easy to read and modify, enabling you to develop custom modules with Python's vast library ecosystem.

Integration with Modern Tools

Ocelot's Python implementation allows scripting beam physics simulations, building GUIs, and integrating ML methods for advanced research.
